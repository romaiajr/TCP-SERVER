## TCP-SERVER

### Setup do projeto
- Na pasta raiz do projeto, executar pip install requirements.txt -r

### Inicializar Broker MQTT
- Baixar o Mosquitto, acessar a pasta raiz e executar mosquitto.exe -v

### Executar servidor
- python server.py

### Executar setor
- python section.py
